```python
def _make_observer(label: str):
    """Create an observer callback that prints transcript lines with a speaker label."""
    def observer(event: dict):
        if event.get("type") == "transcript":
            click.echo(f"[{label}]: {event['text']}", nl=True)
        elif event.get("type") == "agent_token":
            pass  # Skip streaming tokens for terminal output
    return observer
```

2. **Add `_build_goal(goal: str, identity: str) -> str` helper:**
```python
def _build_goal(goal: str, identity: str) -> str:
    """Combine identity and goal into a single goal string for the LLM."""
    if identity:
        return f"You are {identity}. {goal}"
    return goal
```

3. **Add `async def _run_local_call(caller_cfg: dict, callee_cfg: dict) -> None` coroutine:**
   - Import at function scope: `from shuo.services.local_isp import LocalISP` and `from shuo.conversation import run_conversation`
   - Create two LocalISP instances: `isp_caller = LocalISP()` and `isp_callee = LocalISP()`
   - Call `LocalISP.pair(isp_caller, isp_callee)` -- MUST happen before any start()
   - Build goals: `caller_goal = _build_goal(caller_cfg.get("goal", ""), caller_cfg.get("identity", ""))` and same for callee
   - Create two asyncio tasks:
     ```python
     task_caller = asyncio.create_task(
         run_conversation(
             isp_caller,
             observer=_make_observer("CALLER"),
             get_goal=lambda _: caller_goal,
         )
     )
     task_callee = asyncio.create_task(
         run_conversation(
             isp_callee,
             observer=_make_observer("CALLEE"),
             get_goal=lambda _: callee_goal,
         )
     )
     ```
   - Wait for first completion: `done, pending = await asyncio.wait([task_caller, task_callee], return_when=asyncio.FIRST_COMPLETED)`

   - Print summary: `click.echo("\n[CALL ENDED]")`
   - Check for exceptions in done tasks: `for t in done: if t.exception(): raise t.exception()`

The key challenge is mocking `run_conversation` — it is an async function imported inside `_run_local_call`. Mock it at its source: `@patch("shuo.conversation.run_conversation")` or at the cli module's import point.

Since `_run_local_call` does `from shuo.conversation import run_conversation` inside the function body, mock `shuo.conversation.run_conversation`.

For LocalISP, mock `shuo.services.local_isp.LocalISP`.

Both mocks need to be async-compatible:
- `run_conversation` mock: `AsyncMock()` that returns `None`
- `LocalISP.pair` mock: regular `MagicMock()`
- `LocalISP()` mock: return mock instances

Since `local-call` uses `asyncio.run()`, and CliRunner runs synchronously, the Click command's `asyncio.run()` call will execute the mocked coroutine in a real event loop inside the test. The `AsyncMock` for `run_conversation` handles this correctly.

Also mock env vars (DEEPGRAM_API_KEY, GROQ_API_KEY, ELEVENLABS_API_KEY) for tests that should pass the env check.

Example:
```python
from unittest.mock import patch, AsyncMock, MagicMock

@patch("shuo.conversation.run_conversation", new_callable=AsyncMock)
@patch("shuo.services.local_isp.LocalISP")
@patch.dict(os.environ, {"DEEPGRAM_API_KEY": "x", "GROQ_API_KEY": "x", "ELEVENLABS_API_KEY": "x"})
def test_local_call_runs(mock_isp_cls, mock_run_conv):
    runner = CliRunner()
    result = runner.invoke(cli, [
        "local-call",
        "--caller-goal", "ask about balance",
        "--callee-goal", "answer questions",
    ])
    assert result.exit_code == 0
    assert mock_run_conv.call_count == 2
    assert mock_isp_cls.pair.called
```
