    **test_sample_scenarios_pass:**
    This is the critical e2e test. It must run each sample scenario against the real IVR mock server but with a mocked `run_conversation` (since the real agent requires LLM API keys).

    Strategy: Mock `run_conversation` to simulate the agent's behavior for each scenario. The mock must:
    1. Accept the BenchISP as the `isp` parameter
    2. Wait for `_inject` to be set (conversation loop sets this)
    3. For "navigate-to-sales": when the IVR driver injects the menu text, call `await isp.send_dtmf("1")` to simulate the agent pressing 1
    4. For "navigate-to-tech-support": call `await isp.send_dtmf("2")` then `await isp.send_dtmf("1")`
    5. For "timeout-no-input": do nothing (just wait, let timeout handle it)

    Implementation approach:
    - Start the IVR server on an ephemeral port using `_find_free_port()` + threading + uvicorn (or use httpx ASGI transport for in-process testing — the `httpx.AsyncClient(transport=httpx.ASGITransport(app=ivr_app))` pattern avoids needing a real server)
    - If using ASGI transport: modify the test to pass the ASGI client to run_scenario or call IVRDriver.drive directly
    - If using real server: start uvicorn in daemon thread, wait for health, run scenarios, let thread die

    Recommended approach (simpler, more reliable): Use the real IVR server in a daemon thread (same as run_benchmark does). Mock `run_conversation` to simulate agent DTMF responses. Call `run_scenario()` for each sample scenario with the IVR server URL.

    The mock `run_conversation` should be an async function that:
    ```python
    async def _fake_conversation(isp, observer=None, get_goal=None, tts_pool=None, flux_pool=None, ivr_mode=None, **kwargs):
        # Simulate conversation loop: set _inject hook, then respond to IVR events
        event_queue = asyncio.Queue()
        isp._inject = event_queue.put_nowait
        # Wait for IVR driver to inject events, respond with DTMF
        while True:
            try:
                event = await asyncio.wait_for(event_queue.get(), timeout=5.0)
            except asyncio.TimeoutError:
                break
            if hasattr(event, 'transcript') and observer:
                observer({"type": "transcript", "text": event.transcript})
            # Agent decides DTMF based on goal (extracted from get_goal)
            goal = get_goal("") if get_goal else ""
            if "sales department" in goal.lower() and "press 1 for sales" in event.transcript.lower():
                await isp.send_dtmf("1")
            elif "technical support" in goal.lower():
                if "press 2 for support" in event.transcript.lower():
                    await isp.send_dtmf("2")
                elif "press 1 for technical" in event.transcript.lower():
                    await isp.send_dtmf("1")
    ```

    Patch `shuo.bench.run_conversation` with this mock.

    Assert for each scenario:
    - navigate-to-sales: result.passed == True, result.dtmf_log == ["1"], "sales" appears in transcript (case-insensitive)
    - navigate-to-tech-support: result.passed == True, result.dtmf_log == ["2", "1"]
    - timeout-no-input: result.passed == True (max_turns is generous at 20)

    Mark the test with `@pytest.mark.asyncio` since run_scenario is async.
