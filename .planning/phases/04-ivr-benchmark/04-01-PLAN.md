    Create `shuo/shuo/bench.py` with these exact exports:

    ```python
    @dataclass
    class SuccessCriteria:
        transcript_contains: list[str] = field(default_factory=list)
        dtmf_sequence: Optional[str] = None
        max_turns: Optional[int] = None

    @dataclass
    class ScenarioConfig:
        id: str
        description: str
        agent: dict           # {"goal": str, "identity": str}
        success_criteria: SuccessCriteria
        timeout: int = 30
        ivr_flow: Optional[str] = None

    @dataclass
    class CriteriaResult:
        transcript_pass: bool
        dtmf_pass: bool
        turns_pass: bool
        passed: bool          # transcript_pass and dtmf_pass and turns_pass

    @dataclass
    class ScenarioResult:
        scenario_id: str
        passed: bool
        criteria: CriteriaResult
        turns: int
        dtmf_log: list[str]
        transcript: list[str]
        wall_clock_s: float
        error: Optional[str] = None
    ```

    Implement `load_scenarios(path: str) -> list[ScenarioConfig]`:
    - Use `yaml.safe_load()` to read the file
    - Iterate `data["scenarios"]`, parse each into ScenarioConfig
    - Required fields: `id`, `description`, `agent`, `success_criteria`
    - `success_criteria` is a dict parsed into SuccessCriteria
    - Raise `ValueError` with descriptive message if `id` is missing
    - `ivr_flow` and `timeout` are optional (default None and 30)

    Implement `evaluate_criteria(criteria: SuccessCriteria, transcript: list[str], dtmf_log: list[str], turns: int) -> CriteriaResult`:
    - `transcript_pass`: if `criteria.transcript_contains` is empty, True; else all strings must appear (case-insensitive) in `" ".join(transcript).lower()`
    - `dtmf_pass`: if `criteria.dtmf_sequence` is None, True; else `"".join(dtmf_log) == criteria.dtmf_sequence`
    - `turns_pass`: if `criteria.max_turns` is None, True; else `turns <= criteria.max_turns`
    - `passed`: all three are True

    Create `shuo/tests/test_bench.py` with all tests from the behavior block above. Use `pytest` and `tmp_path` fixture for YAML file tests. Import from `shuo.bench`.
